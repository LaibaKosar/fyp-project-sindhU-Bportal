# Required Dependencies for UBAdminDashboard

The Governance Command Center requires the following npm packages:

```bash
npm install recharts react-leaflet leaflet
```

These are needed for:
- **recharts**: Radial and Radar charts for compliance visualization
- **react-leaflet** & **leaflet**: Interactive map for campus distribution

After installation, the dashboard will display:
- Interactive map with university locations
- Radial compliance charts
- Resource radar charts
